게임 소개
**마이티(Mighty)**는 한국의 대표적인 트릭테이킹(trick-taking) 카드 게임입니다. 5명의 플레이어가 참여하며, 비딩을 통해 주공을 정하고 여당(주공+친구)과 야당(나머지 3명)으로 나뉘어 점수 카드를 놓고 대결합니다.

이 프로그램은 1명의 유저와 4명의 AI가 대전하는 싱글 플레이어 웹 기반 마이티 게임입니다.

주요 특징
브라우저만으로 즐길 수 있는 순수 HTML/CSS/JavaScript 구현
전략적 AI 의사결정 시스템
PC 및 모바일 반응형 레이아웃
테스트 모드를 통한 AI 카드 공개 및 의사결정 로그
시각/사운드 이펙트를 통한 생동감 있는 게임 경험
다판 연속 플레이 및 누적 점수 관리
게임 규칙
기본 구성
플레이어: 5명 (유저 1명 + AI 4명)
카드: 53장 (표준 52장 + 조커 1장)
배분: 각 플레이어 10장, 바닥 3장
라운드: 총 10트릭
점수 카드: A, K, Q, J, 10 (총 20장 = 20점)
팀 구성
여당: 주공(master) + 친구(friend)
야당: 나머지 3명(opposition)
친구는 주공이 지정한 "친구 카드"를 가진 사람이며, 해당 카드가 처음 플레이될 때까지 정체가 공개되지 않음
승리 조건
여당 승리: 여당이 획득한 점수 카드 수 ≥ 공약 숫자
야당 승리: 여당이 공약 달성 실패
백런(Backrun): 야당이 점수 카드 20장을 모두 획득 (특별 보너스)
점수 계산
여당 승리 시:

기본 점수 = (여당 획득 점수 - 10) × 배율
주공: +기본점수 × 2, 친구: +기본점수
야당 각자: -기본점수
여당 패배 시:

기본 점수 = (공약 - 여당 획득 점수) × 배율
주공: -기본점수 × 2, 친구: -기본점수
야당 각자: +기본점수
노프렌드 시:

주공: ±기본점수 × 4, 야당 각자: ∓기본점수
배율:

일반: ×1
노기루다: ×2

게임 진행 흐름
Phase 1: 카드 배분 (Dealing)
53장 셔플
바닥에 3장 배치
각 플레이어에 10장씩 배분
풀신청(딜미) 자격 확인
Phase 2: 풀신청 (Deal Check)
다음 조건 중 하나를 만족하면 풀신청 가능:

마이티 보유 + 점수 카드 0장
조커 보유 + 점수 카드 0~1장
점수 카드 0장 (마이티/조커 없음)
풀신청 시 다른 플레이어가 공약 20으로 진행할지 결정

Phase 3: 비딩 (Bidding)
딜러 다음 플레이어부터 시계방향 진행
이전 비딩보다 높은 비딩 또는 패스
비딩 범위: 13~20
같은 숫자일 경우 수트 강도로 비교: ♠ > ◆ > ♥ > ♣ > No
마지막 1명 남으면 주공 확정
전원 패스 시 재배분
Phase 4: 카드 교환 (Exchanging)
주공이 바닥 3장을 가져옴 (13장 보유)
3장을 선택하여 버림
버린 카드의 점수 카드는 주공 점수에 포함
기루다 변경 가능:
일반: 공약 +2 페널티
공약 20: 무료 변경, 노기루다, 솔리노(노프렌드) 가능
Phase 5: 친구 선언 (Friend Declaration)
주공이 친구 카드를 지정:

마이티 친구: 마이티 카드 소유자가 친구
조커 친구: 조커 소유자가 친구
특정 카드 친구: 해당 카드 소유자가 친구
노프렌드: 혼자 진행 (솔리노)
Phase 6: 사연 (Sayeon) — 선택적
주공이 친구에게 조커 보유 여부를 확인:

주공: "사연?"
친구 응답: "넌사" (조커 있음) / "사연" (조커 없음)
Phase 7: 플레이 (Playing)
10라운드에 걸쳐 트릭 진행:

리드 플레이어가 카드 1장 출전
시계방향으로 각 플레이어가 1장씩 출전
트릭 승자가 해당 트릭의 점수 카드 획득
트릭 승자가 다음 리드
Phase 8: 결과 (Game End)
점수 계산 및 누적 점수 갱신
게임 기록 표시
계속 플레이 또는 종료 선택
특수 카드
마이티 (Mighty) 👑
기루다가 스페이드일 때: ◆A (다이아몬드 에이스)
그 외: ♠A (스페이드 에이스)
모든 카드를 이기는 최강 카드
조커콜 당한 트릭에서는 조커 대신 마이티를 낼 수 있음
조커 (Joker) 🃏
2~9라운드에서 마이티를 제외한 모든 카드를 이김
1라운드와 10라운드에서는 무효 (일반 카드 취급)
조커콜에 의해 강제로 내야 할 수 있음
리드 시 원하는 수트를 선언 가능
조커콜 (Joker Call) 📢
기루다가 클로버일 때: ♠3
그 외: ♣3
리드로 낼 때 조커 보유자는 반드시 조커를 내야 함
2~9라운드에서만 유효
특수 규칙
기루다 (Giruda)
비딩 시 선택하는 으뜸 수트
리드 수트가 아닌 기루다 카드로도 트릭 승리 가능 (기루다 컷)
노기루다: 기루다 없이 진행 (점수 배율 2배)
트릭 승리 우선순위
마이티 (항상 최강)
조커 (2~9라운드, 조커콜 아닌 경우)
기루다 수트 (높은 숫자 우선)
리드 수트 (높은 숫자 우선)
카드 출전 규칙
리드 수트와 같은 수트가 있으면 반드시 해야 함
같은 수트가 없으면 아무 카드 가능
마이티는 항상 낼 수 있음
조커는 2~9라운드에서 마이티가 트릭에 없을 때 낼 수 있음
초구(1라운드) 제한:
조커 리드 불가
주공은 기루다 리드 불가 (다른 수트가 있을 경우)
백런 (Backrun)
야당이 점수 카드 20장을 모두 획득
주공: 기본 패널티의 2배, 친구: 기본 패널티
야당: 기본 보너스
AI 시스템
비딩 AI
손패 평가: 수트별 카드 수, 높은 카드, 마이티/조커 보유 여부
예상 트릭 수 기반 비딩 값 결정
노기루다 비딩 판단 포함
보수적 기준으로 무리한 비딩 방지
친구 선언 AI
마이티 미보유 → 마이티 친구 우선
마이티 보유 + 조커 미보유 → 조커 친구
마이티 + 조커 보유 → 기루다 A 또는 K 친구
노프렌드 판단: 마조 동석 + 강한 패 시 솔리노 선언
