# 🎴 마이티 (Mighty)

한국의 전통 트릭테이킹 카드 게임 **마이티**를 웹 브라우저에서 즐길 수 있는 싱글 플레이어 게임입니다.
4명의 AI와 함께 플레이하며, 테스트 모드를 통해 AI의 의사결정 과정을 실시간으로 확인할 수 있습니다.

---

## 📖 게임 소개

마이티는 5명이 53장의 카드(조커 포함)를 사용하여 플레이하는 트릭테이킹 게임입니다.
비딩을 통해 주공이 결정되고, 주공은 친구를 지정하여 여당(주공+친구)과 야당(나머지 3명)으로 나뉘어 대결합니다.

---

## 🎮 게임 규칙

### 기본 구성

| 항목 | 내용 |
|------|------|
| 플레이어 수 | 5명 (유저 1명 + AI 4명) |
| 카드 | 53장 (4수트 × 13장 + 조커 1장) |
| 배분 | 1인당 10장, 바닥 3장 |
| 라운드 | 총 10라운드 (트릭) |
| 점수 카드 | A, K, Q, J, 10 (총 20점) |

### 게임 진행 순서

1. **카드 배분** — 각 플레이어에게 10장, 바닥에 3장
2. **풀신청 확인** — 특수 조건 충족 시 풀신청 가능
3. **비딩** — 13~20 사이의 숫자와 기루다 수트를 선언하여 경매
4. **카드 교환** — 주공이 바닥 3장을 가져오고 3장을 버림
5. **친구 선언** — 주공이 친구 카드를 지정
6. **사연** — 주공이 친구의 조커 보유 여부를 확인 (선택)
7. **카드 플레이** — 10라운드에 걸쳐 트릭 진행
8. **결과 계산** — 여당/야당 점수에 따라 승패 결정

### 비딩

- 최소 비딩: **13**, 최대 비딩: **20**
- 같은 숫자일 경우 수트 강도로 비교: **스페이드 > 다이아 > 하트 > 클로버 > 노기루다**
- 전원 패스 시 카드를 다시 배분
- **노기루다** 비딩 가능 (기루다 없이 진행, 점수 2배)

### 특수 카드

| 카드 | 설명 |
|------|------|
| **마이티** | 가장 강한 카드. 기루다가 스페이드일 때 ◆A, 그 외 ♠A |
| **조커** | 2~9라운드에서 리드 시 수트 선언 가능. 마이티에게만 짐 |
| **조커콜** | 조커를 강제로 꺼내는 카드. 기루다가 클로버일 때 ♠3, 그 외 ♣3 |

### 카드 플레이 규칙

- 리드 플레이어가 수트를 결정하고, 나머지는 같은 수트를 내야 함 (팔로우)
- 해당 수트가 없으면 아무 카드나 낼 수 있음
- **초구(1라운드)에서 주공은 기루다를 리드할 수 없음** (올 기루다 제외)
- 기루다 수트는 리드 수트를 이길 수 있음 (트럼프)
- 마이티는 모든 카드를 이김
- 조커는 2~9라운드에서 마이티를 제외한 모든 카드를 이김
- 1라운드와 10라운드에서 조커는 승리 능력 없음
- 조커콜 카드가 리드되면 조커 보유자는 반드시 조커를 내야 함

### 카드 교환 (주공)

- 주공은 바닥의 3장을 가져온 후 아무 3장을 버림
- **공약 20**: 기루다 변경, 노기루다 선언, 솔리노(노프렌드) 가능 (무료)
- **공약 19 이하**: 기루다 변경 시 공약 +2 페널티

### 친구 선언

- 주공이 특정 카드를 지정하면, 그 카드를 가진 플레이어가 친구
- 친구 카드가 플레이되는 순간 친구가 공개됨
- **노프렌드**: 친구 없이 혼자 플레이 (점수 4배)
- **솔리노**: 공약 20에서 노프렌드 선언 (카드 교환 시 선택)

### 풀신청

다음 조건 중 하나를 만족하면 풀신청 가능:

- 마이티 보유 + 점수 카드 없음
- 조커 보유 + 점수 카드 1장 이하
- 마이티/조커 미보유 + 점수 카드 없음

풀신청이 발생하면 다른 플레이어가 공약 20으로 진행하겠다고 선언해야 하며, 아무도 선언하지 않으면 판이 무효됩니다.

### 사연 (四然)

- 주공이 첫 트릭 전에 친구에게 조커 보유 여부를 질문
- 친구는 진실로 대답해야 함
  - **"넌사"** — 조커 보유
  - **"사연"** — 조커 미보유
- 주공이 조커를 가지고 있거나, 친구가 조커로 지정된 경우에는 사연 불가

### 점수 계산

| 상황 | 주공 | 친구 | 야당(각) |
|------|------|------|----------|
| **여당 승리** | +(여당점수-10)×배율×2 | +(여당점수-10)×배율 | -(여당점수-10)×배율 |
| **야당 승리** | -(공약-여당점수)×배율×2 | -(공약-여당점수)×배율 | +(공약-여당점수)×배율 |
| **노프렌드 여당 승리** | +(여당점수-10)×배율×4 | — | -(여당점수-10)×배율 |
| **노프렌드 야당 승리** | -(공약-여당점수)×배율×4 | — | +(공약-여당점수)×배율 |
| **백런 (야당 20점)** | -20×배율×2 | -20×배율 | +20×배율 |

- **배율**: 노기루다 시 ×2, 그 외 ×1

## 🔧 테스트 모드

테스트 모드를 켜면 다음 기능을 사용할 수 있습니다:

- **AI 카드 공개** — 모든 AI의 손패를 확인
- **AI 의사결정 로그** — AI가 왜 그 카드를 냈는지 실시간 확인
- **낼 수 있는 카드 표시** — 현재 턴 플레이어의 유효 카드 하이라이트

## 🤖 AI 전략

AI는 다음과 같은 전략적 판단을 수행합니다:

### 비딩
- 손패의 수트 분포, 높은 카드, 마이티/조커 보유 여부를 종합 평가
- 예상 트릭 수를 기반으로 적정 비딩 값 결정
- 노기루다 비딩 가능성 평가

### 친구 선언
- 마이티 미보유 시 → 마이티 친구 우선
- 마이티 보유 + 조커 미보유 시 → 조커 친구
- 마이티 + 조커 동시 보유 시 → 기루다 A/K 친구 또는 노프렌드 평가

### 카드 플레이
- **여당**: 보스카드 리드, 기루다 소진, 친구 지원, 판 포기 전략
- **야당**: 마이티/조커 소진 전략, 기루다 소진, 주공 void 수트 공략
- **친구**: 주공 리드 시 무조건 탈환 지원, 주공 void 수트 리드로 기루다 컷 유도
- **카드 추적**: 플레이된 카드, 각 플레이어의 void 수트, 보스카드 판정
